# Rule Baseline — v0.1

## Background
The current classroom check-in entry is easy to ignore and gives weak feedback after completion.

## Goal
Improve learner awareness of check-in progress and make the next expected action clearer.

## Included in this round
- clearer check-in status copy
- stronger completion feedback
- more explicit entry for the next study action

## Not included in this round
- incentive system redesign
- attendance policy changes
- teacher-side management changes

## Stage
This baseline is marked as `review`.
